# 第６回
## ゲーム製作(ex06/game.py)
image.png
### ゲーム概要
- ex06/game.pyを実行すると、1920x1060のスクリーンが表示され、キャラクターをスペースキーでジャンプさせて障害物を避けながら進む横スクロールゲーム
- キャラクターが障害物に接触or穴に落下するとゲーム終了
### 操作方法
- スペースキーでジャンプ
### 追加機能
- 背景のスクロール
- ステージの障害物をリストで管理して表示する
- リストの値によって穴を描画
- 走るキャラクターを描画
- スペースキーでキャラクターがジャンプ
- 障害物（穴）の上にジャンプしていない状態で乗ると死亡判定となり、キャラクターが穴の底に落ちる
